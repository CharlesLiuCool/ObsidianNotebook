## Arc Length as a Function of a Parameter
Let ${{\bf r}(t)}$ describe a smooth curve, for ${t \geq a}$. The arc length is given by
$${s(t) = \int_{a}^t|{\bf v}(u)|du,}$$
where ${|{\bf v}| = |{\bf r}'|}$. Equivalently, ${\dfrac{ds}{dt} = | {\bf v}(t)|}$. If ${|{\bf v}(t)| = 1}$, for all ${t \geq a}$, then the parameter ${t}$ corresponds to arc length.

>[!question] 14.4 (40)
>Determine whether the curve already uses arc length as a parameter. If not, find a description of the curve that *does* use arc length as a parameter
>$${\vec{r}(t) = \langle t^2, 2t^2, 4t^2 \rangle,\;\;\;1\leq t \leq 4}$$

>[!check]- Solution
>${\vec{r}'(t) = \vec{r}(t) = \langle 2t, 4t, 8t \rangle}$
>${|\vec{r}'(t)| = |\vec{v}(t)| = \sqrt{4t^2 + 16t^2 + 64t^2} = \sqrt{84t^2} = 2\sqrt{21}|t| = 2\sqrt{21}t}$
>
>Define ${s(t) = \int_1^t|\vec{r}'(u)|du = \int_1^t(2\sqrt{21}u)du = \sqrt{21}u^2\rvert_1^t = \sqrt{21}(t^2 - 1)}$
> 
> So, on this curve, ${s = \sqrt{21}(t^2 - 1)}$
> ${\implies \dfrac{s}{\sqrt{21}} = t^2 - 1 \implies \dfrac{s}{\sqrt{21}} + 1 = t^2}$
> 
> So ${\vec{r}(t) = \langle t^2, 2t^2, 4t^2 \rangle = \left\langle \dfrac{s}{\sqrt{21}} + 1, 2\left(\dfrac{s}{\sqrt{21}} + 1\right), 4\left(\dfrac{s}{\sqrt{21}}+ 1\right)\right\rangle = \vec{r}(s)}$
> 
> ${\implies \vec{r}(s) = \langle 1, 2, 4 \rangle + s \left\langle \dfrac{1}{\sqrt{21}}, \dfrac{2}{\sqrt{21}}, \dfrac{4}{\sqrt{21}} \right\rangle}$

## Curvature

**Curvature**: Measures how tightly the curve *"bends"* as we move along it.

![[6. 2024-01-30 08.53.08.excalidraw|800]]

The definition:
For a smooth, parametrized curve, the *curvature* is

${K = \left|\dfrac{d\vec{T}}{ds}\right|}$ (${\vec{T}}$ = unit tangent)

= ${\dfrac{\left|\frac{d\vec{T}}{dt}\right|}{\frac{ds}{dt}} = \left| \dfrac{\vec{T}'(t)}{\vec{r}'(t)}\right| = \dfrac{|\vec{T}'(t)|}{|\vec{r}'(t)|}}$
