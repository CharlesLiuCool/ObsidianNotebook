## 14.2,3,4,5 Calculus with Space Curves and Motion in Space (Velocity, Acceleration, etc.)

If ${\vec{r}(t) = \langle x(t), y(t), z(t) \rangle}$, Then

$${\begin{align}& \vec{r}(t) = \dfrac{d}{dt}(\vec{r}(t)) = \dfrac{d\vec{r}}{dt} = \lim_{h\to0} \dfrac{\langle x(t+h), y(t+h), z(t+h) \rangle - \langle x(t), y(t), z(t) \rangle}{h}\\ \\ &=\;... = \langle x'(t), y'(t), z'(t) \rangle \end{align}}$$

(or: ${\dfrac{d}{dt}(\vec{r}(t)) = \dfrac{d\vec{r}}{dt} = \dfrac{d}{dt}\langle x(t), y(t), z(t) \rangle \implies \left\langle \dfrac{dx}{dt} \dfrac{dy}{dt}, \dfrac{dz}{dt} \right\rangle}$

${\vec{r}'(t) = \lim_{h\to0} \left(\dfrac{\vec{r} (t+h) - \vec{r}(t)}{h} \right)}$

Interpreting ${\vec{r}(t)}$ geometrically/physically:
![[5. 2024-01-25 08.22.17.excalidraw]]

So ${\vec{r}'(t)}$ is called the **tangent vector** to ${\vec{r}(t)}$

The **unit tangent vector** to ${\vec{r}(t)}$ (at any t) is:
(recall unit means length 1)

$${\text{unit tangent vector} = \vec{T}(t) = \dfrac{\vec{r}'(t)}{|\vec{r}'(t)|}}$$

In terms of motion in space: 

If ${\vec{r}(t)}$ is the position (vector) for an object as a function of time ${t}$, Then:

${\vec{r}'(t)}$ is the **velocity** of the object, denoted ${\vec{v}(t)}$
${\vec{r}''(t)}$ is the **acceleration** of the object, denoted ${\vec{a}(t)}$
${|\vec{r}'(t)|}$ is the **speed** of the object.

## Definition: Position, Velocity, Speed, Acceleration

Let the **position** of an object moving in the three-dimensional space be given by ${{\bf r}(t) = \langle x(t), y(t), z(t) \rangle}$, for ${t \geq 0}$. The **velocity** of the object is
$${{\bf v}(t) = {\bf r}'(t) = \langle x'(t), y'(t), z'(t) \rangle}$$
The **speed** of the object is the scalar function
$${|{\bf v}(t)| = \sqrt{x'(t)^2 + y'(t)^2 + z'(t)^2}}$$
The **acceleration** of the object is ${{\bf a}(t) = v'(t) = {\bf r}''(t)}$.

>[!check]- Solution
>${\vec{T}(t) = \dfrac{\vec{r}'(t)}{|\vec{r}(t)|} = \dfrac{\langle 1, 0, -\dfrac{2}{t^2} \rangle}{\sqrt{1 + \dfrac{4}{t^2}}}}$

>[!question] 14.3 (19)
>Find velocity, speed, and acceleration for an object whose position is ${\vec{r}(t) = \langle 1, t^2, e^{-t} \rangle}$.

>[!check]- Solution
>${\vec{r}'(t) = \vec{v}(t) = \langle 0, 2t, -e^{-t} \rangle}$
>${\vec{r}''(t) = \vec{v}'(t) = \vec{a}(t) =  \langle 0, 2, e^{-t} \rangle}$

## Theorem 14.1 Derivative Rules

1. ${\dfrac{d}{dt} (c) = 0}$
2. ${\dfrac{d}{dt}(u(t) + v(t)) = u'(t) + v'(t)}$
3. ${\dfrac{d}{dt}(f(t){\bf u}(t)) = f'(t){\bf u}(t) + f(t){\bf u}'(t)}$
4. ${\dfrac{d}{dt}(u(f(t)) = u'(f(t))f'(t)}$

>[!question] 14.3 (50)
>Given that (for some object)
${\vec{a}(t) = \langle t, e^{-t}, 1 \rangle}$, ${\vec{v}(0) = \langle 0, 0, 1 \rangle}$
${\vec{r}(t) = \langle 4, 0, 0}$, find ${\vec{r}(t)}$

>[!check]- Solution
>${\vec{a}(t) = \langle t, e^{-t}, 1 \rangle = \vec{v}'(t)}$
>so ${\vec{v}(t) = \langle \dfrac{t^2}{2} + C, -e^{-t} + D, t + E \rangle}$
> 
> now, ${\vec{v}(0) = \langle 0, 0, 1 \rangle \implies \langle C, -1 + D, E \rangle = \langle 0, 0, 1 \rangle}$
> 
> ${\implies C = 0, D = 1, E = 1}$
> 
> So ${\vec{v}(t) = \langle \dfrac{t^2}{2}, -e^{-t} + 1, t + 1 \rangle}$
> 
> Next, ${\vec{v}(t) = \langle \dfrac{t^2}{2}, -e^{-t} + 1, t + 1 \rangle = \vec{r}'(t)}$

We can write the above procedure like this:
$${\begin{align} \vec{v}(t) &= \int\vec{a}(t)dt \\ \\ &= \int \langle 5, e^{-t}, 1 \rangle dt \\ \\ &= \langle \dfrac{1}{2}t^2, -e^{-t}, t \rangle + \langle C, D, E \rangle \end{align}}$$
