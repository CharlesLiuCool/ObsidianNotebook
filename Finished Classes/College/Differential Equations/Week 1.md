>[!question] Example 1:
>${\dfrac{dy}{dx} = 3y, y = ce^{3x}, c \in \mathbb{R}}$

>[!check]- Solution
>${\dfrac{dy}{dx} = 3ce^{3x},\;3ce^{3x} = 3(ce^3x)}$

>[!question] Example 2:
>${\dfrac{d^2y}{dx^2} = -4y, y = c_1\cos(2x) + c_2\sin(2x)}$

>[!check]- Solution
>${\dfrac{d^2y}{dx^2} = -4(c_1 \cos(2x) + c_2\sin(2x))}$
>
>${\dfrac{dy}{dx} = -2c_1\sin(2x) + 2c_2\cos(2x)}$
> 
> ${\dfrac{d^2y}{dx^2} = -4c_1\cos(2x) - 4c_2\sin(2x) = -4)c_1\cos(2x) + c_2\sin(2x)}$
>
