### Definition 6.

${\{{\bf U}_1, {\bf U}_2 ... {\bf U}_k\}}$ in ${\mathbb{R}^n}$ is called an *orthogonal set* if 
$${{\bf U}_i \cdot {\bf U}_j = 0} \text{ for all } i \neq j$$

### Example 7.

${\{{\bf e}_1, {\bf e}_2 ... {\bf e}_k\} = \left\{ \begin{bmatrix} 1 \\ 0 \\ 0 \end{bmatrix} ,\begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix},\begin{bmatrix} 0 \\ 0 \\ 1 \end{bmatrix}\right\}}$ is an orthogonal set in ${\mathbb{R}^3}$

#### Solution:

Since

$${{\bf e}_1 \cdot {\bf e}_2 = {\bf e}_1 \cdot {\bf e}_3 = {\bf e}_2 \cdot {\bf e}_3 = 0}$$

${\{{\bf e}_1, {\bf e}_2, {\bf e}_3\}}$ is an orthogonal set in ${\mathbb{R}^3}$

### Example 8.

${\{{\bf U}_1, {\bf U}_2 ... {\bf U}_k\} = \left\{ \begin{bmatrix} 2 \\ 0 \\ 0 \end{bmatrix} ,\begin{bmatrix} 0 \\ -3 \\ 0 \end{bmatrix},\begin{bmatrix} 0 \\ 0 \\ 4 \end{bmatrix}\right\}}$ is an orthogonal set in ${\mathbb{R}^3}$

#### Solution:

Since

$${{\bf U}_1 \cdot {\bf U}_2 = {\bf U}_1 \cdot {\bf U}_3 = {\bf U}_2 \cdot {\bf U}_3 = 0}$$
${\{{\bf U}_1, {\bf U}_2, {\bf U}_3\}}$ is an orthogonal set in ${\mathbb{R}^3}$

### Definition 7.

An *orthogonal basis* for a subspace ${H}$ in ${\mathbb{R}^n}$ is a basis that is an orthogonal set.

### Definition 8.

An *orthonormal basis* for a subspace ${H}$ in ${\mathbb{R}^n}$ is an orthogonal basis of unit vectors.

### Remark:

${\{{\bf e}_1, {\bf e}_2, {\bf e}_3\}}$ in Example 7 forms an *orthogonal* basis and *orthonormal* basis for ${\mathbb{R}^3}$

since,
>**1.** ${\large{\{{\bf e}_1, {\bf e}_2, {\bf e}_3\}}}$ is a basis for ${\mathbb{R}^3}$
>**2.** ${{\bf e}_1 \cdot {\bf e}_2 = {\bf e}_1 \cdot {\bf e}_3 = {\bf e}_2 \cdot {\bf e}_3 = 0}$
>**3.** ${||{\bf e}_1|| = ||{\bf e}_2|| = ||{\bf e}_3|| = 1}$

Since condition **1.** and **2.** are satisfied, the basis is *orthogonal*

Since all conditions are satisfied, the basis is *orthonormal*
