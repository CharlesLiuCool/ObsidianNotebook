## *Abstract Data Type*
- An ADT model for data types
- Focuses on what operations can be performed, *NOT* how they are implemented.
- Examples: List, Stack, Queue, etc.

## *ADT vs Data Structure*
- **ADT:** Logical description (what operations are supported)
- **Data Structure:** Concrete implementation (how memory is organized and stored)
