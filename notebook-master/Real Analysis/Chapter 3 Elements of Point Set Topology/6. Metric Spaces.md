## **Metric Spaces**

>**Definition:** A *metric space* is a nonempty set ${M}$ together with a function ${d: M \times M \rightarrow \Bbb{R}}$ satisfying:
>1. ${d(x,y) \ge 0 \;\forall\;x,y \in M}$
>${d(x,y) = 0 \iff x = y}$
>2. ${d(x,y) = d(y,x)}$
>3. ${d(x,y) \le d(x,z) + d(z,y)}$
>Denoted by ${(M,d)}$

In short, its nonnegative, symmetric, and satisfies the triangle inequality.

**Example** ${(\Bbb{R}^n, d)}$
${d(x,y) = ||x - y||}$

**Example** ${(\Bbb{C}, d)}$
${d(z_1, z_2) = |z_1 - z_2|}$

**Example** ${M \neq \emptyset}$ arbitrary nonempty set
${d(x,y) = \begin{cases} 0 & \text{ if } x = y \\ 1 & \text{ if } x \neq y\end{cases}}$

**Example** ${M = \{\underline{x} = (x_1,x_2): x_1^2 + x_2^2 = 1\} = S'}$
${d(\underline{x}, \underline{y}) = }$the length of the smaller arc from ${\underline{x}}$ to ${\underline{y}}$ on the unit circle.

**Example** ${(\Bbb{R}^n, d)}$, ${\underline{x} = (x_1, ...,x_n)\;\;\underline{y} = (y_1, ...,y_n)}$
${d(\underline{x}, \underline{y}) = |x_1 - y_1| + ... + |x_n - y_n|}$

**Example** ${(\Bbb{R}^n, d)}$
${d(\underline{x}, \underline{y}) = \max\{|x_1-y_1|, ..., |x_n - y_n|\}}$
