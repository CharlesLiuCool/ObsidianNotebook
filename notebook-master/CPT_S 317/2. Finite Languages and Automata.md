### **Regular Languages**

- Simplest class of languages
- Represented with finite automata
- We use $R$ to denote the class of all *regular languages*

> Regular language $R$ is a minimal class of languages defined as follows:
> 1. $\emptyset \in R$
> 2. $\{\Lambda\} \in R$
> 3. $\{a\} \in R, a \in \Sigma$
> 4. If $L_1$ and $L_2 \in R$
> 	1. $L_1 \cup L_2 \in R$
> 	2. $L_1L_2 \in R$
> 	3. $L_1^* \in R$

___

### **Regular Expressions**

>*Regular expressions* are defined as
>1. $\emptyset$ is a regExp
>2. $\Lambda$ is a regExp
>3. $a$ is a regExp for each $a \in \Sigma$
>4. If $r_1$ and $r_2$ are regEx then so are
>	1. $r_1 + r_2$
>	2. $r_1r_2$
>	3. $r_1^*$

*Remark:* Each regular expression corresponds to a language (set of string), not a string

### **Alphabets and Symbols**

Let $\Sigma$ be a nonempty and finite set.
$\Sigma$ is called an *alphabet*, where each element is called a *symbol*.

Define $\Sigma^k$: the set of all words on $\Sigma$ with length $k$
e.g. $\Sigma = \{a,b\}$
$\Sigma^2 = \{aa,ab,ba,bb\}$
$\Sigma^1 = \{a,b\}$
$\Sigma^0 = \{\Lambda\}$

### **Bit Strings**

Define $\{0,1\}^n$ as all n-length bit strings.

We can always represent an $n$ size integer with $\log_2(n)$ bits.

### **Operations on Languages**

1. *Union*
$L_1 \cup L_2 = \{w: w \in L_1 \text{ or } w \in L_2\}$

Example: 
$L_1 = \{a, b\}$
$L_2 = \{b, ab\}$

$L_1 \cup L_2 = \{a,b,ab\}$

Note: we also write L_1 + L_2 to denote union

2. *Concatenation*
$L_1 \cdot L_2 = \{xy : x \in L_1 \text{ and } y \in L_2\}$

Example: 
$L_1 = \{a,ab\}$, $L_2 = \{b,ab\}$
$L_1 \cdot L_2 = L_1L_2 = \{ab,aab,abb,abab\}$

Special Case:
$L_1 = \emptyset$, $L_2 = \{b,ab\}$
$L_1L_2 = \emptyset$


3. *Kleene-Star*

$$L^{*} = \bigcup_{k \ge 0} L^k = L^0 \cup L^1 ... \cup L^k$$


$$L^+ = \bigcup_{k \ge 1} L^k = L^1 \cup L^2 ... \cup L^k$$
