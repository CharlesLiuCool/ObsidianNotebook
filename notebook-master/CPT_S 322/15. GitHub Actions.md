### **GitHub Actions**
- *Github Actions* is a powerful automation platform built directly into GitHub, enabling developers to create CI/CD (Continuous Integration and Continuous Deployment) pipelines for their project
- GitHub Actions allows teams to streamline their development processes, automate tasks, and more
- Workflows are triggered by events and define a series of steps for automating various tasks.

### **What is CI/CD**
*Continuous Integration (CI):*
- This is the practice of frequently merging code changes into a central repository, followed by automated builds

*Continuous Development (CD)*
- CD automates the release process automating it
