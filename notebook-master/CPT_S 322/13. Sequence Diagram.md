### **Synchronous and Asynchronous Messages**

- Messages appear in sequential order in the lifeline
- Synchronous messages represented by a solid line with a solid arrowhead. This symbol is used when a sender must wait for a response to a message before it continues the sender wait until the receiver had complete the processing
- Asynchronous, the sender can do whatever even before message is sent. Return message is optional
