### **Understanding Clocks**

- Clock period: duration of a clock cycle
- What's the clock frequency?

$$\text{clock frequency} = 1/\text{clock period}$$

$\text{CPU Time} = \text{clock cycles} \times \text{clock period} = \text{clock cycles} / \text{clock frequency}$

### **Deconstructing Programs**

- Consider `c = a + b`
- We can decompose this into instructions
	- load register `a`
	- load register `b`
	- add `a,b`
	- store `c`

$$\text{CPU Time}_\text{program} = \text{Inst}/\text{Prog} \times \text{Clock}/\text{Inst} \times \text{s}/\text{clock} = \text{s}/\text{Prog}$$

### **Weighted Average CPI**

$$\frac{\sum\limits_{i=1}^2 CPI_i \times n_i \times CP}{\text{Total Instructions}}$$
